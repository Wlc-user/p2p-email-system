#### 4. 消息存储优化（优先级：⭐⭐⭐⭐）
**现状**: JSON文件存储
**需求**: SQLite数据库

**改进**:
```python
# 当前: JSON文件
{"inbox": [], "sent": []}

# 改进: SQLite数据库
CREATE TABLE emails (
    id INTEGER PRIMARY KEY,
    message_id TEXT UNIQUE,
    sender_id TEXT,
    recipient_id TEXT,
    subject TEXT,
    body TEXT,
    encrypted_content BLOB,
    timestamp REAL,
    is_read BOOLEAN
);
CREATE INDEX idx_sender ON emails(sender_id);
CREATE INDEX idx_recipient ON emails(recipient_id);
CREATE INDEX idx_timestamp ON emails(timestamp);
```

**优势**:
- 更快查询（索引优化）
- 支持复杂查询（日期范围、关键词搜索）
- 更安全（加密存储）

预计工作量: 3天

## 📊 竞品分析

### Signal
**优势**: 开源、加密、用户多
**劣势**: 需要手机号、中心化服务器
**我们的优势**: 纯P2P、无需手机号

### ProtonMail
**优势**: 邮件标准、界面友好
**劣势**: 需要服务器、收费贵
**我们的优势**: 无服务器、免费

### Threema
**优势**: 匿名、加密
**劣势**: 收费、中心化
**我们的优势**: 免费、去中心化
