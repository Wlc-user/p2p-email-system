# P2P邮件系统 - 快速启动指南

## 🚀 立即开始（3步）

### 1. 安装Node.js
下载: https://nodejs.org/
推荐版本: v18.x 或 v20.x

### 2. 启动应用
双击运行 `启动P2P应用.bat`

或者手动执行:
```bash
cd p2p-mail-app
npm install
npm run dev
```

### 3. 开始使用
应用会自动打开，你可以:
- ✅ 写邮件
- ✅ 查看收件箱
- ✅ 管理联系人
- ✅ 端到端加密

## 🐛 常见问题

### Q: 应用无法启动？
A:
1. 检查Node.js是否安装: `node --version`
2. 检查端口是否被占用: `netstat -ano | findstr :5173`
3. 删除 `node_modules` 重新安装依赖

### Q: P2P连接失败？
A:
1. 检查端口8100是否开放
2. 检查防火墙设置
3. 确认对方节点已启动

### Q: 邮件发送失败？
A:
1. 确认已与对方交换公钥
2. 确认对方节点在线
3. 检查节点ID是否正确

### Q: 如何获取对方节点ID？
A:
1. 对方启动应用后，在设置页面可以看到节点ID
2. 或者通过DHT网络自动发现
