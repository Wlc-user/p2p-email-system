# P2P邮件系统 - 使用说明

## 快速启动

### 方法1：一键启动（推荐）
双击运行 `启动P2P邮件系统.bat`，会自动启动：
- 后端API服务器（端口 8102）
- 前端界面（端口 5173）

### 方法2：手动启动
1. **启动后端**
   ```bash
   cd "ant coding/p2p"
   python p2p_global.py api
   ```

2. **启动前端**（新开一个终端）
   ```bash
   cd p2p-mail-app
   npm run dev
   ```

3. **访问系统**
   - 前端界面：http://localhost:5173
   - 后端API：http://localhost:8102

## 功能特点

### 1. 联系人管理（人性化设计）

#### 添加单个联系人
1. 点击"添加联系人"按钮
2. 填写信息：
   - **名称**：联系人姓名
   - **节点ID**：40位十六进制字符
   - **分组**：可选（如：同事、朋友、家人）
3. 实时验证节点ID格式
4. 点击"添加"

#### 批量导入联系人
1. 点击"导入"按钮
2. 按照格式输入联系人信息（每行一个）：
   ```
   张三,707c12e8dd7dc34001e5dbe76aabaec89444440c,同事
   李四,fedcba0987654321098765432109876543210fed,朋友
   王五,1234567890abcdef1234567890abcdef12345678,家人
   ```
3. 点击"导入"，系统会自动验证并添加

#### 搜索和筛选
- **搜索**：输入姓名或节点ID的部分字符
- **筛选**：按分组筛选联系人（全部、同事、朋友等）

#### 快捷操作
- **📧 发送邮件**：点击邮件图标直接跳转到写邮件
- **📋 复制节点ID**：点击节点ID区域复制完整ID
- **🗑️ 删除**：点击删除按钮移除联系人

### 2. 节点ID管理

#### 获取自己的节点ID
1. 启动系统后，在顶部状态栏可以看到：
   ```
   🟢 已连接    ID: 9a654e51...  🌍 公网: 123.45.67.89:8000  🔒 端到端加密
   ```
2. 点击"ID: 9a654e51..."区域可复制完整的40位节点ID

#### 分享节点ID
1. 点击顶部状态栏的节点ID区域
2. 系统自动复制到剪贴板
3. 通过其他方式（微信、QQ等）分享给朋友

#### 公网地址
- 当 STUN 成功获取公网地址后，会自动显示公网地址
- 格式：`公网: xxx.xxx.xxx.xxx:port`
- 说明您的节点可以从公网访问

### 3. 写邮件（优化体验）

#### 方法1：从联系人发送
1. 进入"联系人"页面
2. 点击联系人右侧的 📧 邮件图标
3. 自动跳转到写邮件，收件人已填好

#### 方法2：手动输入节点ID
1. 进入"写邮件"页面
2. 输入收件人的40位节点ID
3. 实时验证格式是否正确
4. 填写主题和内容
5. 点击"发送"

#### 粘贴节点ID
- 点击"粘贴"按钮，自动从剪贴板读取并粘贴节点ID
- 系统会自动验证格式
- 如果格式正确，显示绿色对勾

#### 格式验证
- **40位字符**：节点ID必须是40个字符
- **十六进制**：只能包含 0-9 和 a-f（或 A-F）
- 实时提示格式是否正确

### 4. 收件箱

#### 查看邮件
1. 点击左侧导航的"收件箱"
2. 点击邮件查看详情
3. 已读/未读状态自动更新

#### 邮件操作
- **回复**：快速回复当前邮件
- **转发**：转发邮件给其他联系人
- **删除**：删除邮件（本地删除）

### 5. 已发送

- 查看已发送的邮件
- 时间戳显示发送时间
- 支持删除操作

## 节点ID格式说明

### 什么是节点ID？
- P2P网络中每个节点的唯一标识
- 基于SHA256哈希生成
- 格式：40位十六进制字符

### 示例
```
正确的节点ID：
9a654e51c15b2e6d1340fb491940cff5aec0e868
707c12e8dd7dc34001e5dbe76aabaec89444440c
fedcba0987654321098765432109876543210fed

错误的节点ID：
- 少于40位：9a654e51c15b2e6d1340fb491940cff5aec
- 多于40位：9a654e51c15b2e6d1340fb491940cff5aec0e868ab
- 非十六进制：9a654e51c15b2e6d1340fb491940cff5aec0e86G
```

### 获取节点ID
1. 启动系统后自动生成
2. 在顶部状态栏显示（点击复制）
3. 分享给其他用户，他们可以添加你为联系人

## 隐私和安全

### 端到端加密
- 所有邮件内容都经过端到端加密
- 只有收件人可以解密
- 服务器无法查看邮件内容

### 节点ID
- 节点ID是公开信息，可以安全分享
- 不需要保密
- 就像邮箱地址一样

### 网络穿透
- 支持 STUN 协议获取公网地址
- 支持 TLS443 伪装（几乎100%穿透率）
- 即使在NAT网络中也能通信

## 常见问题

### Q1: 添加联系人失败
**A:** 检查节点ID格式：
- 必须是40位字符
- 只能包含 0-9 和 a-f/A-F
- 不能有空格或换行

### Q2: 发送邮件失败
**A:** 可能原因：
1. 后端服务器未启动 → 运行 `启动P2P邮件系统.bat`
2. 节点ID格式错误 → 确认是40位十六进制
3. 对方节点未在线 → 无法投递到离线节点

### Q3: 找不到公网地址
**A:** 正常情况：
- 如果在本地网络测试，不需要公网地址
- STUN 需要能访问外网
- 某些网络环境可能无法获取公网地址

### Q4: 如何导出联系人？
**A:** 目前支持：
- 手动复制联系人信息
- 通过"导入"功能的反向格式准备数据
- 后续版本将添加导出功能
