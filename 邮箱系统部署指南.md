# 企业级CS架构邮箱系统 - 部署与测试指南

## 系统概述

本系统是一个完整的客户端-服务器(CS)架构邮件系统,支持双域名隔离、端到端加密、智能防护等功能。

### 核心特性

✅ **双域名支持** - 两个独立邮件系统可互发邮件
✅ **端到端加密** - ECDH + ChaCha20-Poly1305 加密
✅ **安全防护** - 登录防爆、发送限流、垃圾邮件识别
✅ **邮件撤回** - 5分钟内可撤回已发送邮件
✅ **附件支持** - 图片附件发送与读取
✅ **群发功能** - 批量发送邮件
✅ **智能功能** - 快捷回复推荐、快捷操作提取
✅ **并发支持** - 多客户端同时访问

## 安全设计

### 1. 端到端加密

**加密流程:**
1. 用户注册时生成X25519密钥对
2. 发送邮件时获取接收者公钥
3. 使用ECDH计算共享密钥
4. 使用ChaCha20-Poly1305加密内容
5. 服务器仅存储加密内容

**代码示例:**
```python
from cryptography.hazmat.primitives.asymmetric import x25519
from cryptography.hazmat.primitives.ciphers.aead import ChaCha20Poly1305
import hashlib

# 生成密钥对
private_key = x25519.X25519PrivateKey.generate()
public_key = private_key.public_key()

# 计算共享密钥
shared_secret = my_private_key.exchange(recipient_public_key)

# 加密
key = hashlib.sha256(shared_secret).digest()[:32]
nonce = os.urandom(12)
cipher = ChaCha20Poly1305(key)
ciphertext = cipher.encrypt(nonce, plaintext, None)
```

**安全性保证:**
- ✅ 服务器无法查看邮件内容
- ✅ 数据库泄露也安全
- ✅ 只有发送者和接收者可解密

### 2. 登录防爆

**限流策略:**
- 同一IP 5次失败尝试后封禁15分钟
- 密码错误不返回具体提示
- 使用PBKDF2哈希密码(100000次迭代)

**代码实现:**
```python
def check_login_rate_limit(self, ip):
    now = datetime.datetime.now()

    # 清理过期记录
    cutoff = now - datetime.timedelta(hours=1)
    self.login_attempts[ip] = [
        t for t in self.login_attempts[ip] if t > cutoff
    ]

    # 5次失败封禁15分钟
    if len(self.login_attempts[ip]) >= 5:
        self.blocked_ips[ip] = now + datetime.timedelta(minutes=15)
        return False, "IP已被临时封禁"

    return True, ""
```

### 3. 发送限流

**限流规则:**
- 每小时最多发送50封
- 每分钟最多发送5封

**防止DOS攻击:**
- 频率超限返回429状态码
- 记录限流日志
- IP级别封禁

### 4. 密码保护

**存储安全:**
- 密码使用PBKDF2 + SHA256哈希
- 每个用户独立的16字节盐
- 100000次迭代,暴力破解困难

```python
def hash_password(password: str, salt: bytes = None):
    salt = salt or os.urandom(16)

    kdf = PBKDF2HMAC(
        algorithm=hashes.SHA256(),
        length=32,
        salt=salt,
        iterations=100000
    )

    hashed = kdf.derive(password.encode('utf-8'))

    return base64.b64encode(hashed).decode(), salt
```

### 5. 垃圾邮件识别

**识别特征:**
- 关键词匹配(中奖、优惠等)
- 标题全大写
- 多个感叹号
- 可疑URL模式

**评分机制:**
- 每个特征增加权重
- 分数≥0.7标记为垃圾邮件
- 返回具体原因

**代码示例:**
```python
SPAM_KEYWORDS = ['中奖', '优惠', '免费', '点击', '赢取']

def analyze_spam(subject, body, sender_id):
    score = 0.0
    reasons = []

    text = f"{subject} {body}".lower()

    for keyword in SPAM_KEYWORDS:
        if keyword in text:
            score += 0.3
            reasons.append(f"包含敏感词: {keyword}")

    # 检测钓鱼模式
    if re.search(r'点击.*验证.*账户', text):
        score += 0.5
        reasons.append("疑似钓鱼邮件")

    return score >= 0.7, score, "; ".join(reasons)
```

### 6. 邮件撤回核验

**安全检查:**
1. 验证发送者身份
2. 检查邮件状态(只能撤回已发送)
3. 验证时间限制(5分钟内)
4. 防止重放攻击


## 故障排查

### 问题1: 服务器无法启动
**原因:** 端口被占用
**解决:**
```bash
# 检查端口占用
netstat -ano | findstr :5001

# 更换端口
python mail_server.py --port 5003
```

### 问题2: 数据库锁定
**原因:** 多个进程同时访问
**解决:**
```bash
# 关闭所有服务器进程
taskkill /F /IM python.exe

# 检查数据库文件
sqlite3 email_system.db "PRAGMA integrity_check;"
```

### 问题3: 跨域邮件发送失败
**原因:** 接收者未在当前服务器注册
**解决:**
- 在发送前获取接收者公钥
- 确保两个服务器都有该用户的记录
