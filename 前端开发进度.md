## 启动说明

### 启动后端服务器
```bash
# 双击运行
快速启动.bat

# 或使用PowerShell
python mail_server.py --db-path server_a_data/mail.db --domain mail-a.com --port 5001
python mail_server.py --db-path server_b_data/mail.db --domain mail-b.com --port 5002
```

### 启动前端
```bash
cd p2p-mail-app
npm install
npm run dev
```

### 测试后端
```bash
# 简单测试
python test_simple.py

# 完整测试
python test_mail_system_fixed.py
```

## API端点列表

### 认证
- `POST /api/register` - 用户注册
- `POST /api/login` - 用户登录
- `GET /api/publickey/<node_id>` - 获取公钥

### 邮件
- `GET /api/emails/inbox` - 获取收件箱
- `GET /api/emails/sent` - 获取已发送
- `POST /api/emails/send` - 发送邮件
- `POST /api/emails/recall/<email_id>` - 撤回邮件

### 系统
- `GET /api/health` - 健康检查
- `GET /api/server-info` - 服务器信息
