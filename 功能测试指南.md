# P2P邮件系统 - 功能测试指南

## 🎯 测试前准备

### 1. 启动系统

使用一键启动脚本：
```bash
启动系统.bat
```

或手动启动：
```bash
# 终端1: 启动后端API服务器
cd "ant coding/p2p"
python p2p_global.py api

# 终端2: 启动前端Vite服务器
cd p2p-mail-app
npm run dev:vite
```

### 2. 访问前端

打开浏览器访问：
```
http://localhost:5173
```

## 🐛 已知问题与限制

### 当前版本限制

1. **邮件接收需要对方在线**
   - P2P邮件需要接收方节点在线才能接收
   - 离线消息暂未实现中继存储

2. **删除功能为本地删除**
   - 前端删除只在本地生效
   - 后端API暂未实现持久删除

3. **节点发现需要手动配置**
   - 需要先与对方交换公钥
   - 需要在DHT中注册对方节点

4. **单机测试模式**
   - 当前版本主要用于单机演示
   - 真实P2P通信需要多个节点
