## 🔐 安全特性详解

### 1. 端到端加密
```python
# 密钥交换
private_key = X25519PrivateKey.generate()
shared_secret = private_key.exchange(public_key)

# 对称加密
key = SHA256(shared_secret)[:32]
cipher = ChaCha20Poly1305(key)
ciphertext = cipher.encrypt(nonce, plaintext, None)
```

### 2. 密码保护
```python
# PBKDF2哈希
salt = os.urandom(16)
key = PBKDF2(password, salt, 32, 100000, SHA256)
password_hash = salt + key
```

### 3. 限流保护
```python
# 登录限流
- 5次失败后封禁15分钟
- 按IP地址追踪
- 自动解封

# 发送限流
- 每分钟最多5封
- 每小时最多50封
- 防止DOS攻击
```

### 4. 撤回核验
```python
# 三重验证
1. 身份验证: 只有发送者可以撤回
2. 时间验证: 5分钟内可撤回
3. 状态验证: 只有已发送状态可撤回
```
